### Versioning Scheme

### Feature Naming

`feature/userStoryID`

### Bug Branch Naming

`fix/bugId`

### Methodology

1. Create a Feature/Bug Branch
2. Commit and Push(Remote) on the Same Branch
3. Peer Review the Feature
4. Send MR request to dev Branch
5. Merge to dev branch
6. Send PR to qa Branch
